# Nifty50 Weightage Visualizer

## Overview
This repository contains code and visualizations for the Nifty50 Weightage Visualizer. The visualizer displays the weightage of stocks in the Nifty50 index for the year 2023.

## Features
- Interactive Treemap: The visualizer presents the weightage of each stock using an interactive treemap.
- Sector-wise Analysis: The treemap is organized by sectors, allowing users to explore the weightage of stocks within each sector.
- Data Insights: Hovering over a stock in the treemap provides detailed information about the company, sector, and weightage.

## Usage
To use the Nifty50 Weightage Visualizer, follow these steps:
1. Open the [visualization link](https://aksmundada.github.io/Viz/Nifty50-Weightage-Visualizer.html) in a web browser.
2. Explore the treemap by clicking on the sectors and individual stocks.
3. Hover over a stock to view additional details.

## Repository Contents
- `data.csv`: CSV file containing the weightage data for Nifty50 stocks.
- `Nifty50-Weightage-Visualizer.html`: HTML file with the interactive visualizations.
- `README.md`: This file providing an overview and instructions for the repository.
